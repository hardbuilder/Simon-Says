Tutorial

Let's learn first

Simon Says is a memory game that challenges you to remember and repeat a sequence of colors. Each level adds one new color to the sequence, and you must click the correct order to progress. Make a mistake? You start over!

  • Press the button to start the game
  • A color will flash
  • Your job: click the color and remember the pattern
  • Next color will flash
  • Click the color in the remembered pattern
  • If you make a mistake, just click the button to start over
